# FormBoost — Telegram Mini App

## Структура проекта
```
formboost_bot/
├── bot.py          — Telegram бот (aiogram 3)
├── server.py       — FastAPI сервер + API
├── miniapp/
│   └── index.html  — Mini App фронтенд
├── requirements.txt
└── start.sh
```

## Настройка перед запуском

### 1. В bot.py замени 4 строки вверху:
```python
BOT_TOKEN    = "ВСТАВЬ_ТОКЕН_БОТА"        # от @BotFather
MINI_APP_URL = "https://your-domain.com"  # домен где лежит index.html
ADMIN_ID     = 123456789                   # твой Telegram user_id (узнай у @userinfobot)
KASPI_CARD   = "4400 4301 XXXX XXXX"      # твой номер карты
KASPI_NAME   = "Твоё Имя"                 # имя получателя
```

### 2. В miniapp/index.html замени:
```javascript
// Строка ~240:
else window.open('https://t.me/ВАШ_БОТ','_blank');

// Строка ~175:
// Замени номер карты на свой реальный:
<span class="kcode" onclick="copyText('4400430100000000','Карта скопирована')">4400 4301 XXXX XXXX</span>
// И имя получателя:
Получатель: <b>Твоё Имя</b>
```

### 3. Хостинг Mini App
Mini App должен быть на HTTPS домене. Варианты:
- **Railway.app** — бесплатно, деплой за 5 минут
- **VPS + nginx** — стабильнее для продакшна
- **Vercel** — для статики (только index.html)

На Railway: загрузи проект, в настройках укажи `uvicorn server:app --host 0.0.0.0 --port $PORT`

### 4. В @BotFather:
- `/newbot` — создай бота, получи токен
- `/mybots` → выбери бота → Bot Settings → Menu Button → Set URL → вставь `https://your-domain.com`
- Это добавит кнопку «Открыть приложение» в меню бота

### 5. Твой Telegram user_id:
Напиши @userinfobot — он скажет твой id. Вставь в `ADMIN_ID`.

## Как работает оплата

### Telegram Stars
Пользователь выбирает пакет → Mini App отправляет `tg.sendData()` → бот ловит
и отправляет инвойс через `send_invoice` → пользователь платит Stars →
бот получает `successful_payment` → голоса зачисляются **мгновенно**.

### Kaspi
1. Пользователь выбирает Kaspi → Mini App создаёт транзакцию через `/api/kaspi-init`
2. Показывается номер карты + уникальный код `FB-{id}`
3. После перевода пользователь закрывает Mini App и отправляет скрин боту
4. Бот пересылает скрин тебе (ADMIN) с кнопками ✅/❌
5. Нажимаешь ✅ → голоса мгновенно зачисляются пользователю

### Команды для тебя (как админа):
- `/stats` — статистика: юзеры, доход, запуски

## Генерация ответов ИИ
Для открытых вопросов (текстовые поля) система:
1. Автоматически определяет их при анализе формы (тип 0/1 или нет вариантов)
2. Перед запуском генерирует `N` разных ответов через **Grok (x.ai)**
3. Если Grok недоступен — автоматически переключается на **Gemini**
4. Ответы ротируются: пул из 50 ответов циклически распределяется между всеми отправками
